# Replit Prompt — "إشارة / Ishara" — Jordanian Arabic Sign Language Platform

## Project Overview
Build a full-stack web application in **HTML + CSS + Vanilla JavaScript** (no framework needed, optionally use Vite).
This is a platform for deaf and mute Arabic-speaking users in Jordan.
The site must be fully RTL (right-to-left) with Arabic text as the primary language.
English labels are secondary.

## Pages & Structure

### 1. `/` or `/index.html` — Auth Page (Login / Register)

**Layout:**
- Full-screen background: dark gradient `#1F2937 → #0F4F50 → #389182`
- Floating decorative Arabic sign-hand emoji (large, low opacity, background decoration)

- Logo: circular teal icon + "إشارة" in Arabic + "Ishara" in English
- Tab switcher: "تسجيل الدخول" | "إنشاء حساب"

**Login tab fields:**
- Email input (LTR direction)
- Password input (LTR direction)
- "دخول" button (full width, teal)
- "نسيت كلمة المرور؟" link

**Register tab fields:**
- Full name (RTL)
- Email (LTR)
- Password (LTR)
- Confirm password (LTR)
- "إنشاء الحساب" button (full width, teal)

**On submit:** store user info in `localStorage` as JSON, redirect to `/home.html`

## Animations — Technical Requirements

### Hand SVG animations
```css
@keyframes wave {
  0%, 100% { transform: rotate(-15deg); }
  50% { transform: rotate(15deg); }
}
@keyframes point-up {
  0%, 100% { transform: translateY(0); }
  50% { transform: translateY(-10px); }
}
@keyframes open-close {
  0%, 100% { transform: scaleX(1); }
  50% { transform: scaleX(1.12); }
}
@keyframes shake {
  0%, 100% { transform: rotate(-8deg); }
  25% { transform: rotate(8deg); }
  75% { transform: rotate(-8deg); }
}
```

## LocalStorage Schema

```js
// User profile
localStorage.setItem('ishara_user', JSON.stringify({
  name: 'أحمد',
  email: 'ahmed@example.com',
  joined: '2026-04-30'
}));

// User preferences
localStorage.setItem('ishara_prefs', JSON.stringify({
  fontSize: 16,
  textColor: '#1F2937',
  displayMode: 'normal',  // 'normal' | 'high-contrast' | 'dark'
  sound: true
}));

// Conversations
localStorage.setItem('ishara_conversations', JSON.stringify([
  {
    id: 'conv_001',
    title: 'محادثة مع الطبيب',
    date: '2026-04-28',
    signCount: 12,
    entries: []
  }
]));
```

## Typography
- Font stack: `'Segoe UI', 'Noto Sans Arabic', system-ui, sans-serif`

- RTL: set `direction: rtl; text-align: right;` on body
- English labels: LTR override with `dir="ltr"` attribute or `.ltr` class

## Key UX Requirements
1. All Arabic text must be RTL — use `direction: rtl` everywhere
2. Settings (font size, color, mode) must persist in localStorage and apply immediately on change
3. Auth must check localStorage on home.html load — redirect to index.html if not logged in
4. Animated hands must work without any external library — pure CSS animations + SVG
5. Site must work at 375px (mobile) and 1280px (desktop) widths
6. High contrast mode: swap to white background + black text + thick borders
7. Dark mode: invert section backgrounds while keeping Jordan Horizon palette accents

## DO NOT
- Do not use React, Vue, or any JS framework
- Do not use any external image files
- Do not use video backgrounds
- Do not use Bootstrap or Tailwind (write custom CSS only)
- Do not use any paid APIs
