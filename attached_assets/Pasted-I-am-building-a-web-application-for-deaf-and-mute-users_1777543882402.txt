I am building a web application for deaf and mute users in Jordan using Jordanian Arabic Sign Language.

I want to extend my existing project by adding new features without removing or breaking current functionality. The focus should be on two main aspects:
1) Strong system functionality
2) High-quality, modern UI/UX design

The system should support two-way communication between deaf users and hearing users:

1. Sign Language to Text:
- Use the device camera to capture live video of a deaf user.
- Process the video using my trained AI model (Transformer ST-GCN).
- Convert detected sign language into Arabic text in real time.
- Display predictions clearly using large, readable Arabic text.

2. Speech to Sign Language:
- Use the microphone to capture speech from a hearing user.
- Convert speech to text.
- Map each word to a corresponding sign from my custom dataset (22 classes).
- Display the result as sign language using:
  - animated hand gestures, OR
  - a sequence of sign images/videos for each word.

3. UI/UX Requirements:
- Modern, clean, and high-end design.
- Smooth, intuitive user experience (UX-first).
- Accessibility-focused design for deaf users.
- Large, clear Arabic typography.
- Visual feedback instead of relying on sound.
- Smooth animations and transitions.

- Advanced design style is encouraged, such as:
  - glassmorphism
  - smooth scroll / parallax animations (GSAP-style)

- Organize the interface into clear sections:
  - Live Camera Translation
  - Speech to Sign Translation
  - Sign Dictionary
  - Saved Conversations

4. Technical Integration:
- Backend is already built using FastAPI (running locally on port 8000).
- Use API endpoint:
  - POST /predict/frames for sign recognition
- The frontend must connect directly to this backend.

5. Goal:
Create a real-time intelligent communication platform that bridges deaf and hearing people using AI and sign language.

Important:
I don’t want only a functional system — I also want a very strong, visually impressive UI/UX comparable to modern high-end designs (similar to Dribbble or Apple-style interfaces).
